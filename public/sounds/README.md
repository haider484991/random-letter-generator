# Sound Files for Pictionary Word Generator

This directory should contain the following sound files:

1. `generate.mp3` - A sound effect that plays when a new word is generated
2. `timer-end.mp3` - A sound effect that plays when the timer ends

You can find free sound effects on websites like:
- [Freesound](https://freesound.org/)
- [Mixkit](https://mixkit.co/free-sound-effects/)
- [Zapsplat](https://www.zapsplat.com/)

Make sure to use royalty-free sounds that are appropriate for your application.
